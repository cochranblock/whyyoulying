## Repositioning

`whyyoulying` becomes a general-purpose labor/billing integrity toolkit — usable
by anyone who bills or gets billed for labor. Same detection engine, neutral
framing. The thesis: most billing fraud is drift, sloppy timesheets, or manager
pressure — not premeditation. A self-service detector run *before* invoice
submission eliminates the problem upstream. Give people the tools to not
commit fraud in the first place.

| Audience | How they use it |
|----------|-----------------|
| Contractors / consulting firms | Self-audit before invoicing |
| Customers paying for labor | Verify invoices before paying |
| Internal audit / compliance | Routine payroll-vs-billing review |
| Subcontractor PMs | Check primes aren't double-billing your hours |
| Investigators | Original use case, still supported |

> **Origin note (footnote, not front page):** Originally built for DoD IG and
> FBI fraud investigation per DoDI 5505.02/03 and AG Guidelines. See
> `docs/ORIGIN.md`. The detection rules apply to any labor-billing
> relationship.
